******* Класс IntSummaryStatistics *******

public class IntSummaryStatistics extends Object
                                  implements IntConsumer

Объект состояния для сбора статистики, такой как:
- count - количество,
- min - минимум,
- max- максимум,
- sum - сумма,
- average - среднее значение.

Данный класс предназначен для работы со стримами (хотя и не обязательно).
Например, мы можем вычислить сводную статистику по потоку целых чисел с
помощью:
---------------------------------------------------------------------------
IntSummaryStatistics stats = intStream.
                                collect(IntSummaryStatistics::new,
                                        IntSummaryStatistics::accept,
                                        IntSummaryStatistics::combine);
---------------------------------------------------------------------------

Методы IntSummaryStatistics можно использовать в качестве преобразователя для
исходного стрима.

---------------------------------------------------------------------------
IntSummaryStatistics stats = people.stream()
                .collect(Collectors.summarizingInt(Person::getDependents));
---------------------------------------------------------------------------

Примечание для текущей реализации:
Эта реализация не является потокобезопасной. Однако использовать
Collectors.toIntStatistics() - в параллельном потоке безопасно,
поскольку параллельная реализация Stream.collect() - обеспечивает
необходимое разделение, изоляцию и слияние результатов для
безопасного и эффективного параллельного выполнения. Текущая
реализация не проверяет переполнение суммы.

******* Методы класса IntSummaryStatistics *******

- void accept(int value) - записывает новое значение в сводную информацию;

- void combine(IntSummaryStatistics other) - объединяет состояние другого
  IntSummaryStatistics в текущий;

- double getAverage() - возвращает среднее арифметическое записанных значений
  или ноль, если значений нет;

- long getCount() - возвращает количество записанных значений;

- int getMax() - возвращает максимальное записанное значение или Integer.MIN_VALUE,
                 если значения не были записаны;

- int getMin() - возвращает минимальное записанное значение или Integer.MAX_VALUE,
                 если значения не были записаны;

- long getSum() - возвращает сумму записанных значений или ноль, если значения не
                  были записаны;

- String toString() - возвращает строковое представление объекта;
